# Timestep 민감도 검증 결과

## 목적
dt(timestep)와 substep 수가 시뮬레이션 결과에 미치는 영향 확인.

## 테스트 조건
- 구(R=0.3, density=10, mass=1.131kg) 바닥 위에 놓인 상태
- 10N y방향 힘 적용, 1초 후 위치/속도 측정
- mu_sphere=0.3, mu_ground=0.5 → geomean=0.387
- 이론 vel_y ≈ 5.0 m/s (마찰 고려 시)

## 결과

### dt sweep (substeps=1)

| dt | y_pos | z_pos | vel_y | vel_z | 비고 |
|---|---|---|---|---|---|
| 0.0005 | 2.525 | 0.301 | 5.076 | -0.080 | 기준 (가장 정밀) |
| **0.001** | **2.523** | **0.302** | **5.004** | **0.101** | **안정적** |
| **0.002** | **2.523** | **0.308** | **5.012** | **0.079** | **안정적** |
| 0.005 | 2.533 | 0.303 | 5.229 | -0.480 | vel_z 진동 시작 |
| 0.01 | 2.545 | 0.303 | 5.582 | -1.393 | vel_z 큰 진동 |
| 0.02 | 2.532 | 0.401 | 5.324 | -0.728 | z_pos 부정확 |

### substeps sweep (frame_dt=0.01 고정)

| substeps | physics_dt | y_pos | vel_y | vel_z | 비고 |
|---|---|---|---|---|---|
| 1 | 0.01000 | 2.545 | 5.582 | -1.393 | 불안정 |
| 2 | 0.00500 | 2.533 | 5.229 | -0.480 | 약간 진동 |
| **4** | **0.00250** | **2.523** | **5.022** | **0.052** | **안정적** |
| 8 | 0.00125 | 2.522 | 5.086 | -0.113 | 안정적 |

## 분석

### dt 영향
- **dt <= 0.002**: 결과 수렴, y_pos ≈ 2.52, vel_y ≈ 5.0
- **dt = 0.005**: vel_z에 진동 시작 (-0.48), y방향은 아직 근사
- **dt >= 0.01**: 수직 진동 심각 (vel_z = -1.39), 접촉 불안정
- **dt = 0.02**: z_pos가 0.40 (구가 바닥에서 떠오름)

### substeps 영향
- frame_dt=0.01 + substeps=4 (physics_dt=0.0025) → 안정적
- frame_dt=0.01 + substeps=1 (physics_dt=0.01) → 불안정
- **substeps로 physics_dt를 0.0025 이하로 만들면 안정**

### 권장
- **physics dt <= 0.002** (안정 임계)
- 학습 환경: frame_dt=0.01 + substeps >= 4 권장
- 정밀 시뮬: dt=0.001 이하
