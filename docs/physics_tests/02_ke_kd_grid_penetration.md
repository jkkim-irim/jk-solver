# ke/kd Grid Search - 구-박스 충돌 침투 깊이

## 목적
ke(접촉 강성)와 kd(감쇠) 조합별 구-박스 충돌 시 최대 침투 깊이와 법선력 측정.

## 테스트 조건
- 구(R=0.3, density=10) → 10N 힘으로 박스(half=0.5, density=10)에 충돌
- gap=1.2m, 가속 후 충돌
- ke: [100, 500, 1k, 5k, 10k, 50k, 100k]
- kd: [1, 10, 100, 500, 1000]
- 총 35 조합

## 분석

### 경향
- **ke 증가** → 침투 감소, 충돌 법선력 증가 (더 딱딱)
- **kd 증가** → 침투 급감 (감쇠가 충격 흡수), 법선력 스파이크 발생
- ke 낮고 kd 높으면 → 물렁하지만 감쇠가 충격 흡수 (침투 최소)
- ke 높고 kd 낮으면 → 딱딱하지만 진동/바운스 발생

### 권장 조합
- **일반 용도**: ke=10,000, kd=100 — 침투 9.4mm, 법선력 234N (균형)
- **정밀 접촉**: ke=1,000~5,000, kd=500 — 침투 0.3~2.8mm (최소 침투)
- **피해야 할 조합**: ke=100,000 + kd=500~1000 — 불안정, 과도한 법선력

### 주의
- kd가 너무 높으면 (500+) 법선력 스파이크 (~1000N) 발생
- 이는 MuJoCo의 solref 변환에서 damping ratio가 극단적으로 높아지기 때문
- 실제 학습에서는 ke=10k, kd=100 (기본값)이 가장 안전
