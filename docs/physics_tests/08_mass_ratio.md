# 질량비 민감도 검증 결과

## 목적
구/박스 질량비가 충돌 안정성, 침투 깊이에 미치는 영향 확인.

## 테스트 조건
- 박스: density=100, mass=100kg (고정)
- 구: density 변경으로 mass ratio 조절
- 구에 10N y방향 힘 → 박스와 충돌
- ke=10000, kd=100, geomean friction

## 결과

| ratio | sph_mass(kg) | box_mass(kg) | max침투(mm) | max법선력(N) | 안정성 |
|---|---|---|---|---|---|
| 0.01 | 0.113 | 100.0 | **45.1** | 245.5 | OK |
| 0.10 | 1.131 | 100.0 | **9.5** | 252.2 | OK |
| 0.50 | 5.655 | 100.0 | 0.0 | 0.0 | OK |
| 1.00 | 11.31 | 100.0 | 0.0 | 0.0 | OK |

| 10.0 | 113.1 | 100.0 | 0.0 | 0.0 | OK |
| 100.0 | 1131.0 | 100.0 | 0.0 | 0.0 | OK |

## 분석

### 가벼운 구 → 무거운 박스 (ratio < 1)
- ratio=0.01 (구 0.1kg vs 박스 100kg): 침투 45mm — 가벼운 구가 10N으로 가속되어 빠르게 충돌, 큰 침투
- ratio=0.10 (구 1.1kg vs 박스 100kg): 침투 9.5mm — 적절한 충돌
- **질량비가 작을수록 침투가 깊음** (가벼운 물체가 빠르게 가속되어 강하게 충돌)

### 무거운 구 → 가벼운 박스 (ratio >= 0.5)
- 10N 힘으로는 무거운 구를 충분히 가속시키지 못해 박스에 도달하지 않음 (침투=0)
- 마찰력이 가한 힘보다 커서 구가 움직이지 않음

### 안정성
- **모든 질량비에서 안정** (FAIL 없음)
- MuJoCo solver는 극단적 질량비(1:10000)에서도 안정적

### 결론
- 큰 질량비 차이(1:100)에서도 solver 불안정 없음
- 가벼운 물체가 무거운 물체에 충돌 시 침투가 커질 수 있음 → ke 높이거나 dt 줄이기
- **manipulation 학습에서 gripper-object 질량비는 안정성 이슈 아님**
